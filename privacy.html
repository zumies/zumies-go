プライバシーポリシー / Privacy Policy

最終更新日：2026年7月24日 ／ 運営者：合同会社 KS Planning ／ 対象：Zumies Garage Channel Manager（以下「本ツール」）

1. 概要

本ツールは、合同会社 KS Planning（以下「当社」）が、当社が所有・運営する単一のYouTubeチャンネル @zumies_garage を管理するための、社内用・非公開の自動化ツールです。エンドユーザーや第三者は存在せず、当社のローカル環境でのみ動作します。第三者のデータを収集・表示・販売することはありません。

2. YouTube API Services の利用

本ツールは YouTube API Services を使用します。本ツールを利用することにより、 Google プライバシー ポリシー が適用されます。運営者は YouTube 利用規約 および YouTube API Services 利用規約 に同意しています。

3. アクセス・使用する情報

本ツールは、運営者本人が所有するチャンネルに対してのみ、OAuth 2.0 による本人認証で以下を行います。

  • 運営者自身が制作したオリジナル動画のアップロード、カスタムサムネイルの設定
  • 運営者自身のプレイリストへの追加、固定コメントの投稿・視聴者コメントの管理
  • 運営者自身のチャンネルの公開統計（視聴回数・チャンネル登録者数・コメント数など）の取得

取得した情報は、いずれも運営者自身のチャンネル運営の目的にのみ使用します。他者のデータへのアクセス、スクレイピング、公開表示は一切行いません。

4. 情報の保存と共有

取得した情報およびOAuthトークンは、運営者のローカル環境にのみ保存されます。第三者への提供・販売・公開は行いません。公開統計に基づくデータを保存する場合、その保存期間は取得から36か月を超えません。

5. データの削除・アクセス権の取り消し（削除ポリシー）

運営者は、いつでも Google アカウントのセキュリティ設定（サードパーティ アクセス） から本ツールのアクセス権を取り消すことができます。取り消し後、本ツールは当該データにアクセスできなくなります。ローカルに保存された認証情報・取得データは、運営者がファイルを削除することでいつでも消去できます。

6. 第三者提供

本ツールは、YouTube／Google から取得したいかなる情報も第三者に提供・共有しません。

7. お問い合わせ

本ポリシーに関するお問い合わせ：kanazumis@gmail.com

English summary

This is a private, single-user automation tool that manages the operator's own YouTube channel (@zumies_garage) only, authenticated by the channel owner via OAuth 2.0. It uploads the owner's own original videos, sets thumbnails, manages the owner's own playlists/comments, and reads the owner's own public channel statistics. Its use is subject to the Google Privacy Policy. No third-party data is collected, displayed, sold, or shared. All data stays on the operator's local machine; stored statistics are retained for no more than 36 months. The operator can revoke access at any time via Google Account security settings, after which the tool can no longer access the data. Contact: kanazumis@gmail.com
